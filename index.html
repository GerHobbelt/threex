THREEx three.js extensions vendor.js style

THREEx is a way to write three.js extension ala vendor.js. So as little dependancy as possible. It is modules to help you write three.js stuff, not a framework.

Existing Extensions

Here is a list of the existing extensions. Feel free to contribute your own. Just do a pull request , edit this file, copy an existing line and fit it to your own needs.

threex.rendererstats

 It is a three.js extension to display realtime informations about THREE.WebGLRenderer. Here is a basic example. It is widely inpired from @mrdoob stats.js. It is released under MIT license.

threex.transparency

 It is a three.js extension to easily handle transparency. Here is a basic example. It is released under MIT license.

threex.planets

 It helps you to display all the planets from the solar system. It is on planetpixelemporium. As a cherry on the cake, it has a 'atmosphere material'. Here is a earth example.
